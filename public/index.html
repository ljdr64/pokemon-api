Pokeinfo API

Esta API de Pokémon es una interfaz de programación de aplicaciones (API) que proporciona acceso a información detallada sobre diferentes Pokémon. Esta API sigue el modelo de la conocida PokeAPI: pokeapi.co y permite realizar consultas GET para obtener datos específicos sobre los Pokémon.

Características principales:

Consulta de Pokémon

Obtén información detallada sobre cada Pokémon en /api/v1/pokeinfo/, incluyendo sus habilidades, experiencia base, altura, peso, estadísticas y tipos:



        
Datos en formato JSON:

        
La información se devuelve en formato JSON, lo que facilita su
          integración en aplicaciones web y móviles.
