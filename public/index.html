Pokeinfo API

Esta API de Pokémon es una interfaz de programación de aplicaciones (API) que proporciona acceso a información detallada sobre diferentes Pokémon. Esta API sigue el modelo de la conocida PokeAPI: https://pokeapi.co y permite realizar consultas GET para obtener datos específicos sobre los Pokémon.

Características principales:

Consulta de Pokémon

Obtén información detallada sobre cada Pokémon en https://{api_url}/api/v1/pokemon/, incluyendo sus habilidades, experiencia base, altura, peso, estadísticas y tipos:



        
Consulta de Pokémon-Species

        
Obtén información detallada sobre Pokémon-Species de cada Pokémon en https://{api_url}/api/v1/pokemon-species/,
          Incluyendo su felicidad base, tasa de captura, color, grupos de huevos, cadena de evolución, generación, tasa
          de crecimiento, hábitat, identificación, si es bebé, nombre y forma:


        


        
Datos en formato JSON:

        
La información se devuelve en formato JSON, lo que facilita su
          integración en aplicaciones web y móviles.
