Documentación de la API de Pokémon

Introducción

Esta API proporciona información sobre Pokémon. Puede utilizarse para obtener datos detallados sobre un Pokémon específico.

Base URL

https://{api_url}

Endpoints

Obtener información de un Pokémon específico

GET https://{api_url}/api/v1/pokemon/{id}

- Devuelve información detallada sobre un Pokémon específico identificado por su ID.

Parámetros de la solicitud

  • id: El ID numérico único del Pokémon que se desea obtener.

Respuestas

  • 200 OK: Se devuelve la información del Pokémon solicitado.
  • 404 Not Found: No se encontró ningún Pokémon con el ID especificado.

Ejemplo de solicitud

GET https://{api_url}/api/v1/pokemon/25
